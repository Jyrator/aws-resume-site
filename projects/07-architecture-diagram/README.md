# Project 07 — Architecture Diagrams

## Overview
Professional AWS architecture diagrams created using draw.io with official AWS 2026 icons, and Python-generated diagrams using the Diagrams library.

## Tools Used
- **draw.io** — Visual architecture diagrams with AWS icon sets
- **Python Diagrams library** — Programmatic diagram generation
- **Gemini** — Dark neon diagram generation

## Diagrams Produced
- Portfolio overview architecture
- Per-project architecture diagrams (12 total)
- Dark neon themed versions for LinkedIn and Medium

## Key Concepts Demonstrated
- Architecture documentation best practice
- AWS icon standards and diagram conventions
- Infrastructure visualisation for stakeholder communication
