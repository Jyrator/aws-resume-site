# Project 06 — CloudWatch Monitoring Dashboard

## Overview
A centralised CloudWatch dashboard (ejiro-aws-portfolio) monitoring all portfolio projects — Lambda invocations, API Gateway requests, DynamoDB consumption, and CloudFront traffic.

## Services Used
- **CloudWatch** — Metrics, dashboards, and alarms
- **SNS** — Email notifications on alarm triggers

## Dashboard Widgets
- Lambda invocation counts and error rates
- API Gateway request counts (all APIs)
- DynamoDB read/write capacity consumption
- CloudFront request count and error rate
- EC2 CPU utilisation

## Key Concepts Demonstrated
- Centralised observability across multiple services
- CloudWatch metric filters and alarms
- SNS notification integration
- Proactive monitoring strategy
